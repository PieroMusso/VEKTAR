MAGNITUD Y DIRECCIÓN
PREGUNTA 1
Dado el vector A = (3, 4), ¿cuál es su magnitud?
A) 3
B) 4
C) 5
D) 7
Respuesta correcta: C

PREGUNTA 2
¿Cuál es la fórmula correcta para calcular la magnitud de un vector A = (ax, ay)?
A) ax + ay
B) √(ax² − ay²)
C) √(ax² + ay²)
D) ax² + ay²
Respuesta correcta: C

PREGUNTA 3
Dado el vector B = (-5, 0), ¿cuál es su dirección medida desde el eje X positivo?
A) 0°
B) 90°
C) 180°
D) 270°
Respuesta correcta: C

PREGUNTA 4
Un vector tiene componentes (0, -7). ¿Cuál es su dirección?
A) 0°
B) 90°
C) 180°
D) 270°
Respuesta correcta: D

PREGUNTA 5
Si un vector tiene magnitud 10 y dirección 0°, ¿cuáles son sus componentes?
A) (0, 10)
B) (10, 0)
C) (-10, 0)
D) (0, -10)
Respuesta correcta: B

PREGUNTA 6
Dado el vector C = (1, √3), ¿cuál es su dirección aproximada?
A) 30°
B) 45°
C) 60°
D) 90°
Respuesta correcta: C

PREGUNTA 7
Un vector apunta hacia el segundo cuadrante. ¿Cuál de las siguientes opciones es correcta?
A) ax > 0, ay > 0
B) ax < 0, ay > 0
C) ax < 0, ay < 0
D) ax > 0, ay < 0
Respuesta correcta: B

PREGUNTA 8
Dos vectores tienen la misma magnitud pero diferente dirección. ¿Qué se puede afirmar?
A) Son iguales
B) Son opuestos
C) Son diferentes vectores
D) Son perpendiculares
Respuesta correcta: C

PREGUNTA 9
¿Cuál es la dirección de un vector que apunta exactamente hacia abajo en el plano cartesiano?
A) 0°
B) 90°
C) 180°
D) 270°
Respuesta correcta: D

PREGUNTA 10
Un vector tiene componentes (-3, -3). ¿En qué cuadrante se encuentra su dirección?
A) Primer cuadrante
B) Segundo cuadrante
C) Tercer cuadrante
D) Cuarto cuadrante
Respuesta correcta: C
